# Web3 Tech Stack 2025: Tools dan Framework yang Dipakai

Membangun Web3 applications memerlukan kombinasi tools yang berbeda dari traditional web development. Mari kita breakdown complete stack yang dipakai di production projects.

## Smart Contract Development

### Core Framework

| Framework | Use Case | Market Share |
|-----------|----------|--------------|
| **Foundry** | Best DX, testing, fast compile | ~60% new projects |
| **Hardhat** | Good ecosystem, TypeScript | Legacy favorite |
| **Anchor** | Solana development | Solana ecosystem |
| **Cairo** | StarkNet development | StarkNet ecosystem |

### Foundry Quick Start

```bash
# Install Foundry
curl -L https://foundry.paradigm.xyz | bash
foundryup

# Create project
forge init my-project
cd my-project

# Write contract
forge create src/Contract.sol:MyContract

# Test
forge test
```

### Modern Integration Pattern

```typescript
import { createPublicClient, http } from 'viem'
import { mainnet } from 'viem/chains'
import { abi } from './abi'

// Public client untuk read operations
const publicClient = createPublicClient({
  chain: mainnet,
  transport: http()
})

// Read contract
const totalSupply = await publicClient.readContract({
  address: '0x...',
  abi,
  functionName: 'totalSupply'
})

// Watch events
const unwatch = publicClient.watchContractEvent({
  address: '0x...',
  abi,
  eventName: 'Transfer',
  onLogs: (logs) => console.log(logs)
})
```

### Wallet Connection

```typescript
import { useConnect, useAccount, useDisconnect } from 'wagmi'
import { injected } from 'wagmi/connectors'

function ConnectButton() {
  const { connect } = useConnect()
  const { address, isConnected } = useAccount()
  const { disconnect } = useDisconnect()
  
  if (isConnected) {
    return <button onClick={() => disconnect()}>
      Disconnect: {address.slice(0, 6)}...
    </button>
  }
  
  return <button onClick={() => connect({ connector: injected() })}>
    Connect Wallet
  </button>
}
```

## Testing

### Foundry Testing

```solidity
// test/Contract.t.sol
contract ContractTest is Test {
    function testStaking() public {
        vm.prank(user);
        staking.deposit(100e18);
        
        assertEq(staking.stakes(user), 100e18);
    }
    
    function testFuzzDeposit(uint256 amount) public {
        vm.assume(amount > 0 && amount < 1e18);
        
        vm.prank(user);
        staking.deposit(amount);
        
        assertEq(staking.stakes(user), amount);
    }
}
```
