# Gas Fees Ethereum: Cara Kerja dan Strategi Menghemat

Gas adalah mekanisme pembayaran untuk eksekusi transaksi di Ethereum. Memahami cara kerja gas adalah essential untuk siapapun yang menggunakan Ethereum secara regular.

## Dasar-Dasar Gas

### Apa itu Gas?

Gas adalah satuan yang mengukur computational effort yang dibutuhkan untuk execute operations. Setiap transaksi membutuhkan gas:

```
Base transaction: 21,000 gas
Token transfer: ~65,000 gas
Contract deployment: ~1M+ gas
DEX swap: ~150,000-250,000 gas
```

### Komponen Biaya

```
Total Fee = Gas Used × (Base Fee + Priority Fee)

Where:
- Base Fee: ditentukan oleh jaringan, burned
- Priority Fee: tip untuk validator, incentive alignment
```

### EIP-1559 Impact

Sejak London upgrade (EIP-1559), sebagian gas fee di-burn, membuat ETH bersifat deflationary saat network busy.

```
┌────────────────────────────────────────┐
│              TRANSACTION FEE           │
├──────────────┬─────────────────────────┤
│   BASE FEE   │    PRIORITY FEE         │
│   (burned)   │    (to validator)       │
│   Dynamic    │    Optional (0- max)    │
└──────────────┴─────────────────────────┘
```

## Cara Menghitung Biaya

### Manual Calculation

```typescript
// Calculate transaction fee
const gasLimit = 21000; // standard transfer
const gasPrice = 50; // gwei

const feeInGwei = gasLimit * gasPrice // 1,050,000 gwei
const feeInETH = feeInGwei / 1e9 // 0.00105 ETH
const feeInUSD = feeInETH * ethPrice // ~$3.50 at $3300 ETH

// For contract interaction
const estimatedGas = await provider.estimateGas({
  to: contractAddress,
  data: txData
})
const gasPrice = await provider.getFeeData()
const maxFee = estimatedGas * gasPrice.maxFeePerGas
```

### 4. Optimasi Contract Calls

```typescript

// ❌ Multiple sequential calls
const balance = await token.balanceOf(address)
const allowance = await token.allowance(address, spender)
const lastTx = await token.lastTransferFrom(address)

// ✅ Single multicall

  token.balanceOf(address),
  token.allowance(address, spender),
  token.lastTransferFrom(address)
])
```

### 5. Gas Tokens dan Refunds

```typescript
// GST2 (Gas Token) memungkinkan menyimpan gas untuk later use
// Saat gas murah, mint GST2 (store gas credit)
// Saat gas mahal, release GST2 untuk refund

const gasToken = await GasToken.deploy()
await gasToken.mint({ gasPrice: low }) // Saat murah
await gasToken.free(gasAmount) // Saat mahal, dapat refund
```
