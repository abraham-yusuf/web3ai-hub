### Check Eligibility

```typescript
// Via LayerZeroscan (official checker)
import axios from 'axios'

async function checkEligibility(address: string) {
  const response = await axios.get(
    `https://api.layerzerolabs.io/v1/claim/eligibility/${address}`
  )
  
  if (response.data.eligible) {
    console.log(`Eligible for ${response.data.allocation} ZRO`)
    console.log(`Claim window: ${response.data.claimStart} - ${response.data.claimEnd}`)
  } else {
    console.log('Not eligible')
  }
}
```

## Troubleshooting

### Common Issues

| Issue | Solution |
|-------|----------|
| "Not eligible" | Check different wallets, might be on L2s |
| Gas too high | Wait for lower gas or claim from L2 |
| Claim not loading | Clear cache, try different browser |
| Transaction stuck | Speed up/cancel di MetaMask |
| "Already claimed" | Check transaction history |
