# DEX dan AMM: Bagaimana Decentralized Exchange Bekerja

Decentralized Exchanges (DEX) adalah protokol yang memungkinkan trading token secara langsung dari wallet tanpa perlu perantara. Berbeda dengan CEX yang menggunakan order book, DEX menggunakan Automated Market Maker (AMM).

## Apa itu AMM?

AMM adalah smart contract yang menentukan harga berdasarkan mathematical formula, bukan order book. Liquidity providers (LPs) mendepositikan aset ke pool untuk memfasilitasi trading.

### Constant Product Formula (x × y = k)

Ini adalah formula paling fundamental, pertama kali digunakan oleh Uniswap:

```solidity
// Simplified AMM pricing
function getAmountOut(uint256 amountIn, uint256 reserveIn, uint256 reserveOut) 
    public pure returns (uint256) {
    
    uint256 amountInWithFee = amountIn * 997; // 0.3% fee
    uint256 numerator = amountInWithFee * reserveOut;
    uint256 denominator = reserveIn * 1000 + amountInWithFee;
    
    return numerator / denominator;
}
```

**Prinsip:**
- Total liquidity (k) tetap konstan
- Ketika x naik, y harus turun
- Ini menentukan harga secara matematis

```solidity

    function executeLongTermOrders(uint256 tokenIn, uint256 tokenOut, uint256 amountPerBlock) {

    }
```

### 2. Concentrated Liquidity (V3 evolution)
LPs dapat menyediakan liquidity di specific price ranges, meningkatkan capital efficiency.

### 3. Native ETH
Trading langsung dengan ETH tanpa WETH wrapping, saving gas.

## DEX Trading Strategies

### 1. Slippage Management
```typescript
// Set appropriate slippage
const slippage = 0.5; // 0.5%
const minAmountOut = amountOut * (1 - slippage)
```

### 2. MEV Protection
- Gunakan protected RPC (Flashbots RPC)
- Enable MEV protection di wallet settings
- Consider batch auctions untuk large orders

### 3. Route Optimization
```typescript
// Check multiple DEXes for best price
const quoteUniswap = await router.getQuote(tokenIn, tokenOut, amount)
const quoteSushi = await sushiRouter.getQuote(tokenIn, tokenOut, amount)
```

## DEX Security Considerations

1. **Approve carefully** — Set approval only when needed
2. **Verify contracts** — Selalu check contract address di explorer
3. **Beware of phishing** — Bookmark official sites
4. **Start small** — Test dengan amount kecil dulu
5. **Check fees** — Gas + protocol fee + price impact
