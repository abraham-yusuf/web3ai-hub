# Membaca Blockchain Explorer

Blockchain explorer membantu kita melacak aktivitas on-chain secara transparan.

## Explorer Populer

- **Etherscan** untuk Ethereum & EVM chain
- **Solscan** untuk Solana
- **BscScan** untuk BNB Chain

## Informasi Utama di Halaman Transaksi

1. **Tx Hash**: ID unik transaksi.
2. **From / To**: alamat pengirim dan penerima.
3. **Value**: jumlah aset yang dipindahkan.
4. **Gas Used**: biaya eksekusi transaksi.

## Membaca Halaman Address

Di halaman address, kamu bisa melihat:

- saldo aset
- riwayat transaksi
- token yang dimiliki
- interaksi smart contract

## Tips Praktis

- Gunakan fitur “Token Transfers” untuk cek pergerakan token.
- Cek “Contract” untuk membaca source code jika tersedia.
- Bookmark address penting untuk monitoring portofolio.
