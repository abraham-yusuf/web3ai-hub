# Wallet Security 101

Wallet adalah pintu masuk aset digital Anda. Security adalah prioritas utama.

## Checklist wajib

1. Simpan seed phrase offline.
2. Jangan pernah membagikan private key.
3. Gunakan wallet terpisah untuk eksperimen.
4. Verifikasi URL sebelum connect wallet.

> Dalam dunia on-chain, satu klik yang salah bisa berakibat permanen.
