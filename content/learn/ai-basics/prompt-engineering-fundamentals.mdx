# Prompt Engineering Fundamentals

Prompt adalah instruksi yang kita berikan ke model AI. Prompt yang baik membuat output lebih akurat.

## Formula sederhana

Gunakan struktur: **Role + Context + Task + Constraints + Output Format**.

## Contoh

- Role: "Kamu editor teknologi"
- Context: "Artikel untuk pembaca pemula"
- Task: "Jelaskan smart contract"
- Constraints: "Maksimal 300 kata"
- Output: "Gunakan bullet list"
