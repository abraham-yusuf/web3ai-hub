# LLM API Primer

Saat integrasi ke aplikasi, kita biasanya memanggil endpoint model lewat HTTP API.

## Komponen request umum

- model
- messages / prompt
- temperature
- max tokens

## Streaming vs non-streaming

- **Non-streaming**: jawaban dikirim setelah selesai.
- **Streaming**: jawaban dikirim bertahap agar UX terasa lebih cepat.
