# Evaluasi Model AI

Model yang bagus bukan hanya akurat, tapi juga konsisten dan aman digunakan.

## Metrik Umum

- **Classification**: accuracy, precision, recall, F1
- **Regression**: MAE, RMSE
- **LLM**: win rate, human eval, task success rate

## Validasi dan Cross-Validation

Validasi membantu memastikan model tidak overfitting. Gunakan cross-validation saat data terbatas.

## Evaluasi di Produksi

- Monitor drift data
- Catat error dan feedback user
- A/B testing untuk iterasi

<Callout type="info">
  Evaluasi bukan fase sekali jalan — lakukan terus-menerus setelah launch.
</Callout>
